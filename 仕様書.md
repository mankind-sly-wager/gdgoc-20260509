# TOEIC Task Manager 仕様書

## 1. アプリ概要

### 名称（仮）
TOEIC Task Manager

### 目的
TOEIC 学習者が「今日やるべきこと」を迷わず実行できるように、  
**タスク管理・学習計画自動生成・メモ管理・進捗可視化**を提供する Web アプリ。

### 特徴
- Google Workspace のような直感的 UI
- Google Keep 風のメモカード
- Google カレンダー風の進捗可視化
- LocalStorage によるデータ永続化
- TOEIC 学習に特化した学習計画自動生成アルゴリズム

## 3. 状態管理仕様（LocalStorage）

### 保存キー
`toeicState`

### データ構造
```js
{
  tasks: [
    { id, title, meta, completed }
  ],
  memos: [
    { id, title, content, tag }
  ],
  settings: {
    currentScore,
    targetScore,
    examDate
  }
}
```

### 初期値
- タスク3件  
- メモ3件  
- 設定（現在500点 → 目標800点、試験日は90日後）

## 4. 主要機能仕様

### 4.1 ダッシュボード（dashboard）

#### 表示内容
- 今日のタスク達成率
- 完了数 / 全タスク数
- 試験日までの残日数
- 予測スコア推移グラフ（Chart.js）

#### グラフ仕様
- 折れ線グラフ
- ラベル：開始時 → 中間1 → 中間2 → 中間3 → 目標
- スコアは現在値から目標値までを4分割して算出

---

### 4.2 タスク管理（tasks）

#### 表示内容
- 今日のタスク一覧
- 完了チェックボックス
- 削除ボタン
- タスク追加モーダル

#### 操作
- `toggleTask(id)`：完了切り替え
- `deleteTask(id)`：削除
- `submitNewTask()`：追加

---

### 4.3 メモ管理（memos）

#### 表示内容
- Google Keep 風カードレイアウト
- タイトル / 内容 / タグ
- 削除ボタン
- メモ追加モーダル

#### 操作
- `deleteMemo(id)`：削除
- `submitNewMemo()`：追加

---

### 4.4 設定（settings）

#### 入力項目
- 現在のスコア
- 目標スコア
- 試験日

#### 操作
- `saveSettingsOnly()`：保存のみ
- `saveAndGenerateTasks()`：保存＋タスク自動生成

## 5. 学習計画自動生成アルゴリズム

### 入力
- currentScore
- targetScore
- examDate

### 計算ロジック
1. スコア差を算出  
2. 残日数を算出  
3. 必要学習時間を推定  
   - **100点上げるには 250時間必要**
4. 1日あたりの必要時間を算出  
5. スコア帯でカテゴリ配分を変更

### スコア < 600 の場合
- 語彙：40%
- 文法：40%
- リスニング基礎：20%

### スコア ≥ 600 の場合
- Part7 読解：40%
- シャドーイング：40%
- 難単語：20%
